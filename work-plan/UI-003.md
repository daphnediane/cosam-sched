# Add dark mode / light mode switch

## Summary

Implement theme switching with dark, light, and CosAm color modes.

## Status

Open

## Priority

Medium

## Description

Users want the option to switch between dark mode, light mode, and the default Cosplay America color scheme.

## Implementation Details

- Create CSS variables for all three themes
- Add a theme switcher UI component
- Store preference in localStorage
- Ensure all UI elements work in all themes
- Default to CosAm colors, remember user choice
