Cosplay America Schedule

Interactive calendar widget demo — using early 2026 data and editor

Embedding in Squarespace

Upload cosam-calendar.css, cosam-calendar.js, and your schedule.json to a hosting service (or Squarespace file storage). Then add a Code Block with:

<link rel="stylesheet" href="URL/cosam-calendar.css"> <div id="cosam-calendar"></div> <script src="URL/cosam-calendar.js"></script> <script> CosAmCalendar.init({ el: '#cosam-calendar', dataUrl: 'URL/schedule.json' }); </script>

Replace URL with the actual URL where you host the files.
