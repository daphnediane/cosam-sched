## Description

Edge commands add or remove relationships between entities using
`EditContext::add_to_field_cmd` / `remove_from_field_cmd`.

```text
cosam-modify --file <path> [--select <type> <query>] add-edge <edge-field> <target-query>
cosam-modify --file <path> [--select <type> <query>] remove-edge <edge-field> <target-query>
```

### Edge field resolution

`E::field_set().half_edges().find(|d| d.matches_name(name))` locates the
`HalfEdgeDescriptor`. Only `Owner`-side edges support direct add/remove; `NonOwner` edges
return an error with a hint to use the owner side.

### Target lookup

The target entity type is extracted from `half_edge.field_type().item_type()` →
`FieldTypeItem::EntityIdentifier(type_name)`. A match on `type_name` dispatches to the
correct `lookup_list::<TargetType>()` call.
