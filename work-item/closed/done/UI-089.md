# UI-089: Simplify pricing filter to All / Included / Premium

## Summary

The cost filter has too many options; collapse "Additional Cost" and
"Workshops" into a single "Premium" option.

## Status

Completed

## Priority

Medium

## Description

The current filter has four options: All, Included, Additional Cost,
Workshops. The distinction between "Additional Cost" and "Workshops" is not
obvious to attendees and the label "Cost" is ambiguous. Simplify to three
options: All, Included (badge-covered events), Premium (any event that
requires separate payment or registration). Rename the filter label from
"Cost" to "Pricing".

## Acceptance Criteria

- Filter shows three chips: All, Included, Premium.
- "Included" shows only events where `isFree` is true.
- "Premium" shows events where `isFree` is false (covers both workshops and
  additional-cost events).
- The filter group label reads "Pricing".
- Saved/URL state values "included" and "premium" replace the old
  "included" / "paid" / "workshop" values; old values default to "all".
