# Rooms should list both the room name and the hotel room

## Summary

Display both the programming room name (e.g., "Programming 1") and the actual hotel room location.

## Status

Completed

## Priority

Medium

## Description

Currently only the programming room names are shown. Attendees need to see both the programming designation and the actual hotel room number/location for easier navigation.

## Implementation Details

- Update room data structure to include hotel room information
- Modify converter to extract hotel room data from spreadsheet
- Update UI to display both names in room filters and event details
- Consider format: "Programming 1 (Grand Ballroom A)"
