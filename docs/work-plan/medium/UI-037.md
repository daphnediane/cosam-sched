# Display room hours separately from schedule grid

## Summary

Add a dedicated room hours section to display operating hours for rooms with RH/Is Room Hours events, formatted by day and room type as shown in the example layout.

## Status

Open

## Priority

Medium

## Description

Room-hours events (RH prefix or "Is Room Hours" flag) currently appear in the main schedule grid, but they represent operating hours rather than specific panels. These should be displayed separately in a more readable format that shows when each area is open.

## Expected behavior

- Room-hours events displayed in separate section from main schedule
- Grouped by room/area name and organized by day
- Special handling for "CONVENTION OPERATIONS HOURS" (earliest to latest each day)
- Clean, readable format matching the provided example

## Implementation Details

### Data Processing

1. **Extract room-hours events during conversion**:

   ```perl
   sub _extract_room_hours ($events) {
       my %room_hours;
       
       for my $event (@$events) {
           next unless $event->{is_room_hours};
           
           my $room_name = $event->{room_name} // $event->{panel_type};
           my $date = _extract_date($event->{start_time});
           my $start_time = _extract_time($event->{start_time});
           my $end_time = _extract_time($event->{end_time});
           
           push @{$room_hours{$room_name}{$date}}, {
               start_time => $start_time,
               end_time => $end_time,
               event_name => $event->{name}
           };
       }
       
       return \%room_hours;
   }
   ```

2. **Process convention operations hours**:

   ```perl
   sub _calculate_convention_hours ($room_hours) {
       my %convention_hours;
       
       for my $date (keys %{$room_hours->{'CONVENTION OPERATIONS'}}) {
           my @times = @{$room_hours->{'CONVENTION OPERATIONS'}{$date}};
           
           my $earliest = (sort map { $_->{start_time} } @times)[0];
           my $latest = (sort map { $_->{end_time} } @times)[-1];
           
           $convention_hours{$date} = {
               start_time => $earliest,
               end_time => $latest
           };
       }
       
       return \%convention_hours;
   }
   ```

### Widget Display

1. **Create room hours component**:

   ```javascript
   function renderRoomHours(room_hours) {
       const container = document.createElement('div');
       container.className = 'room-hours-section';
       
       // Sort rooms by priority (Registration, Market Expo, etc.)
       const sorted_rooms = sortRoomHoursByPriority(room_hours);
       
       for (const [room_name, days] of sorted_rooms) {
           if (room_name === 'CONVENTION OPERATIONS HOURS') {
               container.appendChild(renderConventionHours(days));
           } else {
               container.appendChild(renderRoomHoursSection(room_name, days));
           }
       }
       
       return container;
   }
   ```

2. **Convention hours special handling**:

   ```javascript
   function renderConventionHours(days) {
       const section = document.createElement('div');
       section.className = 'room-hours-area convention-hours';
       
       const title = document.createElement('h3');
       title.textContent = 'CONVENTION OPERATIONS HOURS';
       section.appendChild(title);
       
       const sorted_days = sortDaysChronologically(Object.keys(days));
       
       for (const day of sorted_days) {
           const hours = days[day];
           const line = document.createElement('div');
           line.className = 'room-hours-line';
           line.textContent = `${day}: ${hours.start_time} – ${hours.end_time}`;
           section.appendChild(line);
       }
       
       return section;
   }
   ```

3. **Regular room hours display**:

   ```javascript
   function renderRoomHoursSection(room_name, days) {
       const section = document.createElement('div');
       section.className = 'room-hours-area';
       
       const title = document.createElement('h3');
       title.textContent = room_name;
       section.appendChild(title);
       
       const sorted_days = sortDaysChronologically(Object.keys(days));
       
       for (const day of sorted_days) {
           const day_events = days[day];
           for (const event of day_events) {
               const line = document.createElement('div');
               line.className = 'room-hours-line';
               line.textContent = `${day}: ${event.start_time} – ${event.end_time}`;
               section.appendChild(line);
           }
       }
       
       return section;
   }
   ```

4. **Day sorting logic**:

   ```javascript
   function sortDaysChronologically(days) {
       const day_order = ['Thursday', 'Friday', 'Saturday', 'Sunday'];
       return days.sort((a, b) => {
           return day_order.indexOf(a) - day_order.indexOf(b);
       });
   }
   ```
