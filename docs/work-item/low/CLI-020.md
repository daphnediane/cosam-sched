# cosam-modify: CLI Editing Tool

## Summary

CLI tool for making batch edits to schedule data from the command line.

## Status

Open

## Priority

Low

## Description

`cosam-modify` provides command-line access to the edit system for scripted
or batch modifications to schedule data.

### Commands

- `cosam-modify set <file> --entity <uuid> --field <name> --value <val>`
- `cosam-modify add-panel <file> --uid <uid> --name <name> [--type <type>]`
- `cosam-modify add-presenter <file> --name <name> [--rank <rank>]`
- `cosam-modify assign <file> --panel <uid> --presenter <name>`
- `cosam-modify schedule <file> --panel <uid> --room <id> --start <time> --duration <min>`
- `cosam-modify find <file> --query <text>` — search entities by text
- `cosam-modify list <file> --type <entity_type>` — list all entities of a type
- `cosam-modify diff <file1> <file2>` — show differences between two schedules
- `cosam-modify jump-start <archive> --from <year> --to <year>` — create new year

### Features

- All mutations go through the edit command system (FEATURE-010)
- Batch mode: read commands from a file or stdin
- Dry-run mode for previewing changes
- JSON output mode for scripting

### Dependencies

- `schedule-data` for the data model and edit system
- `clap` for argument parsing

## Acceptance Criteria

- Can set individual fields on entities
- Can add entities and relationships
- Find returns correct matches
- Jump-start creates a new year correctly
- All edits are reversible (history preserved in output file)
- Integration tests for common workflows
