# cosam-convert: Format Conversion Tool

## Summary

CLI tool for converting between schedule file formats (XLSX, JSON, widget JSON).

## Status

Open

## Priority

Low

## Description

`cosam-convert` is a command-line application for importing and exporting
schedule data between supported formats.

### Commands

- `cosam-convert import <input.xlsx> -o <output.schedule>` — XLSX to internal
- `cosam-convert export-widget <input.schedule> -o <output.json>` — to widget JSON
- `cosam-convert export-xlsx <input.schedule> -o <output.xlsx>` — to XLSX
- `cosam-convert export-xlsx <input.schedule> --update <existing.xlsx>` — update in-place
- `cosam-convert info <file>` — display file metadata and entity counts
- `cosam-convert validate <file>` — check for data integrity issues

### Options

- `--year <N>` — select year for multi-year archives
- `--format <version>` — target format version for widget JSON
- `--verbose` — detailed progress output
- `--dry-run` — for export/update, report what would change

### Dependencies

- `schedule-data` for the data model
- `clap` for argument parsing
- XLSX and JSON format modules from `schedule-data`

## Acceptance Criteria

- Import from XLSX produces a valid internal schedule file
- Export to widget JSON matches expected format
- Export to XLSX round-trips correctly
- `info` and `validate` commands work
- Integration tests with fixture files
