# cosam-convert: Format Conversion Tool

## Summary

CLI tool for converting between schedule file formats (XLSX, JSON, widget JSON).

## Status

Open

## Priority

Low

## Blocked By

- FEATURE-029: XLSX spreadsheet export

## Description

`cosam-convert` is a command-line application for importing and exporting
schedule data between supported formats.

### Commands

- `cosam-convert import <input.xlsx> -o <output.schedule>`
- `cosam-convert export-widget <input.schedule> -o <output.json>`
- `cosam-convert export-xlsx <input.schedule> -o <output.xlsx>`
- `cosam-convert info <file>` — display metadata and entity counts
- `cosam-convert validate <file>` — check data integrity

## Acceptance Criteria

- Import from XLSX produces a valid internal schedule file
- Export to widget JSON matches expected format
- Export to XLSX round-trips correctly
- Integration tests with fixture files
