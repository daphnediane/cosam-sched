# CLI-095: cosam-modify delete command

## Summary

Implement the `delete` subcommand to soft-delete an entity by name or UUID.

## Status

Open

## Priority

Low

## Blocked By

- CLI-092: cosam-modify list and get commands

## Description

`delete` removes the selected entity using `EditContext::remove_entity_cmd` + `apply`,
which soft-deletes via the `__deleted` CRDT flag.

```text
cosam-modify --file <path> [--select <type>] delete <query>
```

An explicit non-wildcard query is required. Bulk `delete *` is intentionally disallowed
without a `--force` flag to prevent accidental mass deletion.

## Acceptance Criteria

- [ ] Deleting by name removes the entity from subsequent `list` output
- [ ] Deleting by UUID removes the entity
- [ ] `delete *` without `--force` returns an error
- [ ] Undo restores the deleted entity (within the same invocation)
