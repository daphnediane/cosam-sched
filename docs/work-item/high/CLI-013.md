# Migrate cosam-convert to schedule-data

## Summary

Port cosam-convert from schedule-core to schedule-data for XLSX-to-JSON conversion.

## Status

Not Started

## Priority

High

## Description

cosam-convert currently uses schedule-core for XLSX import and JSON/HTML export. Migrate it to use schedule-data's `ScheduleFile` API, entity model, and export pipeline. This is the primary CLI tool for producing the schedule widget's JSON.

## Implementation Details

- Replace schedule-core data types with schedule-data entities
- Use schedule-data `ScheduleFile::load_auto()` for XLSX import
- Use schedule-data display export for widget JSON generation
- Use schedule-data full export for internal JSON (V11)
- Port HTML template generation if still needed, or delegate to Perl converter
- Maintain all existing CLI flags and output paths
- Validate output against existing schedule-core output for parity

## Acceptance Criteria

- cosam-convert produces identical widget JSON from the same XLSX input
- All existing CLI flags work as before
- No dependency on schedule-core remains
- Performance comparable to schedule-core version
