# Visual Conflict Indicators in Widget

## Summary

Add visual indicators to the schedule widget to highlight conflicting panels.

## Status

Not Started

## Priority

Medium

## Description

Display conflict data from JSON output in the widget. Conflicting panels should have visual cues (icons, colored borders), hover tooltips with details, and click-to-highlight related conflicts.

## Implementation Details

- Conflict-specific CSS classes: red border for room conflicts, orange for presenter conflicts
- Conflict badge/count in top-right corner of panel card
- Hover tooltip showing conflict details (conflicting panel names and times)
- Click to highlight related conflicting panels and dim others
- Optional conflict summary panel showing all conflicts in current view
- Different visual treatments for different conflict types

## Acceptance Criteria

- Conflicting panels clearly marked with visual indicators
- Hover tooltips show detailed conflict information
- Click to highlight related conflicting panels
- Responsive design works on mobile devices
- Accessible: screen reader support for conflict states
