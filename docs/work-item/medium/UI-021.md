# Sticky Headers and Room Hours Display in Widget

## Summary

Add sticky day/time headers and a separate room hours section to the schedule widget.

## Status

Not Started

## Priority

Medium

## Description

Two related widget improvements: (1) sticky or repeated headers so users maintain day/time context while scrolling the grid, and (2) a dedicated room hours section that extracts room-wide events (registration, market expo, etc.) from the main grid into a separate readable display.

## Implementation Details

### Sticky Headers

- Implement CSS `position: sticky` with proper z-index for day/time headers
- Or insert repeated header rows between time blocks
- Ensure headers don't interfere with panel content
- Test scroll behavior on different devices and screen sizes

### Room Hours Display

- Extract room-hours panels (is_room_hours flag) from main grid
- Display in separate section grouped by room/area and organized by day
- Special handling for convention operations hours (earliest to latest each day)
- Room filter should exclude rooms with only room-hours panels
- Room-hours panels still visible in grid if user toggles them on

## Acceptance Criteria

- Users always see current day/time context while scrolling
- Room-hours panels displayed in a clean separate section
- Room filter excludes rooms with only room-hours panels
- Responsive design works on mobile
